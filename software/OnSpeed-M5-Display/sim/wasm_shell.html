The real OnSpeed M5Stack display firmware, compiled to WebAssembly and rendered to an HTML5 canvas via SDL2. Data is a synthetic AOA ramp (0 → 25° and back) so you can watch the tone-map regions light up in order. Click the canvas first to focus it, then:

  • ← — Button A (dim)
  • ↓ — Button B (cycle display mode: AOA indexer → attitude → narrow → decel → G history)
  • → — Button C (brighten)
