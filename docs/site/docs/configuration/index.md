# Configuration

After installing the hardware, you need to configure the OnSpeed system for your specific aircraft. All configuration is done through a **web browser** — connect to the OnSpeed WiFi network and open the configuration pages.

## Configuration Pages

- **[Web Interface](web-interface.md)** — How to connect and navigate the web UI
- **[First-Time Setup](first-time-setup.md)** — Step-by-step checklist for initial configuration
- **[Sensor Calibration](sensor-calibration.md)** — Zero the pressure sensors and IMU
- **[Flap Position Setup](flap-setup.md)** — Configure flap detection
- **[Audio Settings](audio.md)** — Volume, mute threshold, 3D audio, Vno chime
- **[Advanced Settings](advanced.md)** — AHRS algorithm, smoothing, CAS curve
- **[Backup & Restore](backup-restore.md)** — Save and load configuration files
